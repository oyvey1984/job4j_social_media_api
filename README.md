# Social Media API

RESTful API для социальной медиа платформы.
Приложение позволяет пользователям регистрироваться, публиковать посты, подписываться друг на друга, добавлять в друзья и обмениваться сообщениями.

Проект реализован на **Java + Spring Boot** и предоставляет HTTP API для клиентских приложений (web / mobile).

---

# 📌 Основные возможности

## 👤 Аутентификация и авторизация

* регистрация пользователей
* вход в систему
* безопасное хранение паролей (хэширование)
* авторизация через **JWT**
* защита API через **Spring Security**

---

## 📝 Управление постами

Пользователь может:

* создавать посты
* прикреплять изображения
* редактировать свои посты
* удалять свои посты
* просматривать посты других пользователей

---

## 👥 Взаимодействие пользователей

### Подписки

Пользователь может подписываться на других пользователей.

### Друзья

Механика друзей реализована через заявки:

1. Пользователь отправляет заявку.
2. До принятия заявки он остается **подписчиком**.
3. Если заявка принята — пользователи становятся **друзьями**.
4. Если заявка отклонена — пользователь остается **подписчиком**.

### Удаление из друзей

Если один пользователь удаляет другого из друзей:

* он **также отписывается**
* второй пользователь **остается подписчиком**

---

## 💬 Сообщения

Друзья могут отправлять друг другу сообщения.

Особенности:

* чат в реальном времени **не требуется**
* переписка запрашивается через API

---

## 📰 Лента активности

Лента пользователя содержит:

* последние посты пользователей,
* на которых он подписан.

Поддерживается:

* пагинация
* сортировка по дате создания поста
